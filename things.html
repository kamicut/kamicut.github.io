THINGS.JS

ThingsJS is a toolkit for the internet of things. You can connect a fitbit, an Arduino or potentially any internet connected object to a web app with a simple API.

The way it works is that you can write html tags such as <led pin="6"></led> and it would connect that DOM object to the arduino pin. On the arduino, you simply need to upload the thingsJS script that communicates with the server. Using polymer, websockets and hardware glue code, acrylc and I built a demo for the process. You can view the project on github.
